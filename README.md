# Skills


This repository is a collection of robust skills designed to give LLMs thorough, up-to-date information on packages that may not be as widely represented in mainstream AI training data. The goal is to empower the community to use AI effectively with their favorite niche tools.

## Available Skills

### NodeJS, Typescript and Web

- [Hapi.js](https://hapi.dev/) - A powerful Node.js framework for building applications and APIs
- [Google zx](https://google.github.io/zx/) - A tool for writing shell scripts using JavaScript
- [HTMX](https://htmx.org/) - High-power tools for HTML, enabling modern UIs without heavy JavaScript
- [Joi](https://joi.dev/) - The most powerful schema description language and data validator for JavaScript

### SQL

- [MSSQL Server](https://www.microsoft.com/en-us/sql-server) - T-SQL syntax, SQL Server administration, query performance, stored procedures, indexes, and more
- [SQL Writing Guidelines](https://www.microsoft.com/en-us/sql-server) - My own methodology for writing type-safe, access-controlled SQL Server application databases
- [SQL Server Performance](https://www.microsoft.com/en-us/sql-server) - Diagnosing and fixing T-SQL performance problems, index tuning, and execution plans
- [SQL BI & Reporting](https://www.microsoft.com/en-us/sql-server) - Analytical T-SQL patterns for aggregations, time series, pivots, and data export
- [SQL ML Features](https://www.microsoft.com/en-us/sql-server) - Extracting ML-ready feature datasets from SQL Server for model training
